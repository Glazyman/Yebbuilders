# Before & after videos

Place **8 MP4 files** in this folder using these names (rename from your originals):

| Your file      | Save as here      |
|----------------|-------------------|
| Before1.mp4    | `before-1.mp4`    |
| Before 2.mp4   | `before-2.mp4`    |
| Before 3.mp4   | `before-3.mp4`    |
| Before 4.mp4   | `before-4.mp4`    |
| After1.mp4     | `after-1.mp4`     |
| After 2.mp4    | `after-2.mp4`     |
| After 3.mp4    | `after-3.mp4`     |
| After 4.mp4    | `after-4.mp4`     |

Pairing: `before-1` + `after-1`, … `before-4` + `after-4`.

Spaces in filenames are removed for clean URLs. If you prefer to keep your exact names, update the `<source>` paths in `index.html` under the Before & After section.
